# Sterlington Injectable Pharmaceutical UK Ltd

**Website:** https://sterlingtonpharma.com

B2B supplier of sterile injectable and oral pharmaceutical products. Based in Edinburgh, Scotland, UK.

## Stack
- Pure HTML/CSS/JavaScript — single file
- Hosted on Cloudflare Pages
- Auto-deploys from GitHub main branch

## How to make changes
1. Open `index.html` in Cursor
2. Ask Claude to make the change
3. `git add . && git commit -m "description" && git push origin main`
4. Site updates live in ~60 seconds
